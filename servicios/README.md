# Sitio — Marcos Oyarzo | Ciberseguridad Ciudadana, Educación Digital y Tecnología Consciente

Sitio estático (HTML/CSS/JS) enfocado en **servicios**: charlas, talleres, diagnósticos y asesorías (personas, instituciones y PYMEs),
más una línea de seguridad del hogar con domótica y privacidad.

## Ejecutar localmente
- Abrir `index.html` en el navegador, o
- VS Code + Live Server.

## Personalización recomendada (30–60 min)
1. **Servicios**: ajusta nombres/alcances y agrega casos o temarios.
2. **Recursos**: reemplaza PDFs por tus versiones finales.
3. **Contacto**: cambia el email (si aplica) y agrega una solución de formulario real (Formspree/EmailJS).

## Publicar en GitHub Pages
1. Crea un repo (ej. `marcos-oyarzo-site`).
2. Sube todos los archivos.
3. Settings → Pages → Deploy from a branch → `main` / root.
4. Publica y enlaza tu dominio cuando lo tengas.

## Nota de ética y alcance
El sitio enfatiza enfoque preventivo y educativo. No se entrega contenido ofensivo ni procedimientos ilícitos.
En hogar/domótica: asesoría y configuración; no monitoreo tipo central ni promesas de “seguridad total”.
