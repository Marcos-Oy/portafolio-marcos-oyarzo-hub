# Estudio Web — Portafolio/Servicios

Sitio tipo landing + catálogo para vender desarrollo web a:
- Personas Profesionales (portafolios / marca personal)
- PyMEs (sitio oficial + catálogo)
- Empresas (corporativo escalable)

## Características
- Modo Light/Dark con 1 clic (localStorage + prefers-color-scheme)
- Hero Slider XL (autoplay, flechas, dots, swipe)
- Catálogo de plantillas con filtro por segmento + modal "Ver demo"
- Secciones: Servicios, Plantillas, Proyectos, Clientes, Testimonios, Sobre mí, FAQ, Contacto
- 100% responsivo (móvil/tablet/escritorio)

## Ediciones rápidas
- Nombre / textos: `index.html`
- WhatsApp: buscar `wa.me/56912345678`
- Email: buscar `hola@tusitio.cl`
- Colores: `css/styles.css` (`:root` y `html[data-theme="dark"]`)
- Plantillas: `index.html` sección `#templates` + `js/main.js` objeto `templateMeta`
- Imágenes: reemplazar `.svg` dentro de `/img/` por fotos reales manteniendo nombres o actualizando `src`

## Ejecutar
Abrir `index.html` o usar Live Server (VS Code).
